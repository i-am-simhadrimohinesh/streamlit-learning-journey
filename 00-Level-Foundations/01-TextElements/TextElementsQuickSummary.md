## 🎨 UI & Text Elements Reference 🚀
```
📊 Quick Summary

🏷️ st.logo()       → App branding (logo shown in sidebar/header)
🎯 st.title()      → Main application title
📌 st.header()     → Major section heading
🔹 st.subheader()  → Sub-section / nested heading
🏷️ st.badge()      → Small label/tag for status, category, or highlighting
✍️ st.write()      → Smart display (auto-renders text, data, lists, DataFrames, etc.)
🧾 st.text()       → Plain unformatted text output
📝 st.markdown()   → Rich text formatting using Markdown
📎 st.caption()    → Small muted helper text (notes, timestamps, extra information)
💻 st.code()       → Displays code with syntax highlighting
🧮 st.latex()      → Renders mathematical equations (LaTeX)
🌐 st.html()       → Renders custom HTML inside the app
➖ st.divider()    → Horizontal line to separate UI sections

🧠 Key Idea → Use the right tool for the right purpose:

✨ Formatting → markdown  
📄 Simple text → text  
⚡ Smart output → write  
🧱 Structure → header / subheader / divider  
🏷️ Highlight → badge  
🧮 Math → latex  
```
