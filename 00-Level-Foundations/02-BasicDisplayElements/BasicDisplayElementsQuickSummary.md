## 📊 Basic Display & Debug Elements Reference 🚀
```
📌 Quick Summary

📊 st.metric()     → Displays key performance indicators (KPIs) with optional delta changes
📦 st.json()       → Pretty-prints and visualizes JSON / dictionary data in expandable format
📚 st.help()       → Shows documentation (docstring) of Python objects/functions
👨‍💻 st.echo()       → Displays code and its output together for learning/debugging
⚠️ st.exception()  → Displays full error traceback in a clean formatted way
🚨 st.warning()    → Shows warning messages to alert users without stopping execution

🧠 Key Idea → Use these for analytics, debugging, and user awareness:

📊 KPIs → metric  
📦 Data inspection → json  
📚 Understanding tools → help  
👨‍💻 Teaching/debugging → echo  
⚠️ Safe error display → exception  
🚨 Alerts → warning    
```
